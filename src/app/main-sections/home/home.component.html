Who I Am and Why Nohadon Files?

My name is Pol Gonzalo, I'm a Java / Kotlin Software Engineer and I'm currently working in Barcelona.

Nohadon Files is my digital portfolio. My motivation is to share and document each step of my projects for 2 reasons: to show my professional skills and also to learn and grow my knowledge, as I believe that each piece of code, each algorithm, each framework or engine is a valuable learning experience. Here, you’ll find everything from small technical pieces of code to large projects where I strive to innovate. I particularly enjoy OOP with ( Java / Kotlin / C++ ), graphics development and Game development, exploring the finer details of how technology can become a top-tier visual and functional experience.



What is this page about?

The main purpouses of Nohadon Files are: to show my main professional skills with Java / Kotlin development, and to learn about new Technologies, Languages, Algorithms, etc.

It’s a space where I share my passion for software development and innovation through my projects. Here, you'll find everything from experimental concepts and small programming projects to large-scale developments that push the boundaries of what’s possible with current technology. I am a software developer and game developer too, and each section of this site reflects my dedication to these topics.
